# Privacy and LGPD

## Principios

- Minimizar coleta.
- Controlar acesso por perfil.
- Registrar consentimentos relevantes.
- Permitir exportacao de dados do tenant.
- Anonimizar/excluir quando aplicavel.
- Evitar PII sensivel em logs.

## Papel do SaaS

O SaaS pode atuar como operador para dados dos clientes finais dos tenants e como controlador para dados de relacionamento comercial com os tenants. Contratos e DPA devem formalizar cada caso.
